Second test of vacuum forming table.

This is the second test of the vacuum molding table in c-base. This table was built by ihmis-suski and me from November 2014 to March 2015. We are using 1mm Polystyrol. The machine uses sheets of 1m by 1m and uses 230V/32A/7,2KW to heat the plastics up.
